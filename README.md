# atlas contracts

in the atlas project, a person's "contract" is a statement of who they are (when they are at their best). it takes the form: "i am a(n) [adjective 1], [adjective 2], [adjective 3] leader!"

in this repo, we compute some statistics regarding these adjectives.

the raw data is in `data/`, which is in `.gitignore` for privacy. anonymized data is offered here, both raw and cleaned. notes on the cleaning process are available as comments in `clean.py`.

see [report.md](report.md) for a detailed statistical analysis.
